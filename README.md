# Robotic-arm-control-program-Arduino-and-Processing-
Программное обеспечение для управления роботизированной рукой на основе любительских сервоприводов с применением Arduino, блютуз-модуля и драйвера.
Для сборки проекта требуется программное обеспечение Arduino IDE и Processing.  
Дополнительная информация о работе находится в [Вики репозитория](https://github.com/SemiKairin/Robotic-arm-control-program-Arduino-and-Processing/wiki).
